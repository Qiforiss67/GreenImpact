## Slide 2: Project Overview
### What is GreenImpact?
- **Interactive SDG Platform** - Track contributions to UN's 17 Sustainable Development Goals
- **Role-Based System** - Admin dashboard for management, user interface for participation
- **Community Engagement** - Leaderboards, achievements, and progress tracking
- **Gamified Experience** - Points system and milestone recognition

### Key Objectives
- Promote awareness of Sustainable Development Goals
- Enable individuals to track their environmental impact
- Create a community of environmentally conscious users
- Provide tools for activity management and progress monitoring

## Slide 3: Technologies Used

### Frontend Stack
- **React 18** - Modern UI library with hooks and context
- **Tailwind CSS** - Utility-first styling framework
- **React Router v6** - Client-side routing and navigation
- **Context API + useReducer** - Global state management

### Backend Stack
- **Node.js + Express.js** - Server-side JavaScript runtime and framework
- **JWT Authentication** - Secure token-based authentication
- **bcryptjs** - Password hashing and security
- **Express Validator** - Input validation and sanitization

### Development Tools
- **Git** - Version control system
- **ESLint** - Code quality and formatting
- **Nodemon** - Development server with hot reload
